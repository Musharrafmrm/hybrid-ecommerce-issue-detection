# hybrid-ecommerce-issue-detection
Hybrid ML for E-Commerce Review Issue Detection - 20APC5036

**Supervisor:** Mr. Y.M.S. Tharaka

This project focuses on detecting product and service-related issues from e-commerce reviews using a hybrid machine learning approach.

## Project Structure

- data/        → Dataset storage
- notebooks/   → Google Colab notebooks
- models/      → Trained ML models
- results/     → Outputs, graphs, reports
- thesis/      → Project proposal and documentation

## Objective

To analyze customer reviews and identify key issues using ML techniques.
